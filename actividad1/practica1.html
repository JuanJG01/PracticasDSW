¿QUE ES HTML?

HTMLson las siglas de (hypertext markup Lenguaje),que puede traducirse como leguaje de marcas o marcado de hipertexto

El lenguaje de HTML se emplea para crear las paginas web. Es muy facil ver el codigo HTML de una pagina web, la opciom exacta cambia de un navegador a otro y tambien puede cambiar de una versiom a otra de un mismo navegador,

El lenguaje HTML sirve para describir la estructura basica de una pagina y organizar la forma en que se mostrara su contenido, ademas de que HTML permite incluir enlaces hacia otras paginas o documentos.

HTML se compone de etiquetas que se escriben entre los simbolos menor que y mayor que.

Los inicios del lenguaje HTML se remonta al año 1990, cuando Tim Berners-Lee creo la primera pagina web.
